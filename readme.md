# Курсовий проект: Онлайн-сервіс бронювання спортивних залів

## Технологічний стек

Проект реалізовано з використанням стеку **MERN** та супутніх бібліотек:

* **Backend:** Node.js, Express.js
* **Frontend:** React.js, Tailwind CSS
* **Database:** MongoDB (з використанням Mongoose ODM)
* **Auth:** JWT (JSON Web Tokens)
* **Admin Panel:** AdminJS + Mongoose Adapter

## Архітектура та Патерни

У проекті реалізовано наступні шаблони проектування (GoF):

* **Facade (`AppServiceFacade`):** Надає єдину точку входу для отримання даних каталогів (зали, секції, тренери).
* **Strategy (`DiscountContext`):** Розрахунок вартості послуг залежно від типу абонемента (Premium, Standard тощо).
* **Observer (`LoyaltyObserver`):** Автоматичне нарахування бонусів після успішного бронювання.
* **Singleton (`EventManager`):** Керування подіями в системі через єдиний екземпляр менеджера.
* **Adapter (`AdminJSMongoose`):** Інтеграція адміністративної панелі AdminJS із базою даних Mongoose.

## Інструкція із запуску

Для запуску проекту локально вам знадобиться:

1.  **Node.js** (версії 18 або вище)
2.  **MongoDB** (встановлена локально або посилання на хмарний кластер MongoDB Atlas)

### 1. Налаштування та запуск серверної частини (Backend)

1.  Відкрийте термінал та перейдіть у папку сервера:
    ```bash
    cd backend
    ```

2.  Встановіть необхідні залежності:
    ```bash
    npm install
    ```

3.  Створіть файл конфігурації `.env` у папці `backend`. Додайте туди наступні параметри (приклад):
    ```env
    PORT=5000
    MONGO_URL=mongodb://localhost:27017/sportbooking
    JWT_SECRET=vash_sekretniy_kluch
    ```

4.  **Наповнення бази даних (Seeding):**
    Якщо ви запускаєте проект вперше або ваша база даних порожня, запустіть скрипт для створення початкових даних (зали, тренери, секції):
    ```bash
    node utils/seed.js
    ```

5.  Запустіть сервер:
    ```bash
    npm start
    ```
    Сервер запуститься на порту **5000** (або тому, який ви вказали в `.env`). Якщо все добре, ви побачите повідомлення `MongoDB connected` та `Server running...`.

### 2. Налаштування та запуск клієнтської частини (Frontend)

1.  Відкрийте новий термінал (не закриваючи попередній) та перейдіть у папку клієнта:
    ```bash
    cd frontend
    ```

2.  Встановіть залежності:
    ```bash
    npm install
    ```

3.  Запустіть React-додаток:
    ```bash
    npm start
    ```
    Додаток автоматично відкриється у браузері за адресою [http://localhost:5173](http://localhost:5173).
